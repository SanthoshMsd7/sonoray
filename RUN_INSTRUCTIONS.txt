# ERP PROJECT - RUN INSTRUCTIONS

Follow these steps to get the project running on your local machine.

## PREREQUISITES
1. Node.js (v18 or higher)
2. MySQL Server (Running on default port 3306)

---

## 1. BACKEND SETUP
Open a terminal and navigate to the backend folder:
`cd backend`

1. Install dependencies:
   `npm install`

2. Configure Database:
   - Ensure your MySQL server is running.
   - The connection string is already set in the .env file.
   - Push the database schema:
     `npx prisma db push`

3. Generate Super Admin (Optional - first time only):
   - Start the server: `npm run dev`
   - Use the Setup script at: `http://localhost:5000/api/auth/setup-superadmin`
   - Default credentials created:
     - Email: admin@admin.com
     - Password: admin

4. Run Backend:
   `npm run dev`
   (Server will run on http://localhost:5000)

---

## 2. FRONTEND SETUP
Open a NEW terminal and navigate to the frontend folder:
`cd frontend`

1. Install dependencies:
   `npm install`

2. Run Frontend:
   `npm run dev`
   (App will run on http://localhost:3000)

---

## 3. ACCESSING THE APP
1. Open your browser and go to: http://localhost:3000
2. Log in with:
   - Email: admin@admin.com
   - Password: admin

---

## TROUBLESHOOTING
- If "Unauthorized Role" appears, ensure you are logged in as the Admin (admin@admin.com).
- If the Map doesn't show up, ensure you have an active internet connection (to load Leaflet tiles).
- If the Chat fails, check that both Backend and Frontend terminals are running without errors.
